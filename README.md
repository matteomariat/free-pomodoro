## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Usage

- Click **Start** to begin a 25-minute work session
- Click **+5 min** to extend the current session by 5 minutes (only when paused)
- Click **Reset** to restart the timer
- The timer automatically switches between work and break sessions
- Complete tomatoes appear as you finish sessions
